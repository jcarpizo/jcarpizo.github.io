Database¶

Caution

The Character-Encoding MUST BE set to UTF-8!

1. Naming¶

Type Naming
Entity Singular, lower case, underscore for spacing
Attribute All lower case, underscore for spacing
Relationship Entity Combination of both singular entity names, spaced with an underscore (e.g. account_address)

Example:

Table: account

Attribute Type
id int
name varchar(50)
mail varchar(100)
is_active bit
creation_date datetime

Table: account_address

Attribute Type
id int
account_id int
street varchar(50)
zip char(5)
city varchar(50)
country_id int

2. ORM¶

Note

Documentation: http://docs.doctrine-project.org/projects/doctrine-orm/en/latest/

  • Native-SQL-Queries MUST NOT be used!
