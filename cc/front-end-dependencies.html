Front-End-DependenciesÂ¶

Danger

Don't commit the dependencies to the SVN-System!

All Front-End-Dependencies MUST NOT be installed manually! We use always the bower-installer for this task.

Define all required dependencies in the bower.json and provide the required files in the install section.

Here is an example for a definition file, that requires the dependencies:

  • Bootstrap
  • Jquery
  • Bootstrap-Dialogs
{
  "dependencies": {
    "bootstrap": "~3.3.4",
    "jquery": "~2.1.3",
    "bootstrap3-dialog": "~1.34.4"
  },
  "install": {
    "path": "assets/components",
    "sources": {
      "bootstrap": "bower_components/bootstrap/dist/**",
      "jquery": "bower_components/jquery/dist/**",
      "bootstrap3-dialog": [
        "bower_components/bootstrap3-dialog/dist/css/bootstrap-dialog.min.css",
        "bower_components/bootstrap3-dialog/dist/js/bootstrap-dialog.min.js"
      ]
    }
  }
}

To installl the dependencies call this command in your CLI:

bower-installer
