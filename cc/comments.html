CommentsÂ¶

  • One Space after comment open tag is REQUIRED
  • MUST start with a capital letter
  • MUST end with a full stop
  • Perl-style comments (#) MUST not be used
  • PHP-Doc-Comments MUST not be used

Example of invalid comments:

<?php

# Invalid perl style comment.

/**
 * PHP doc comments MUST NOT be used
 */

//Space is missing.

// must start with a capital letter.

// MUST end with a full stop

Example of valid comments:

<?php

/*
 * Multi
 * line
 * comment
 */

// Must start with a capital letter and MUST end with a full stop.
