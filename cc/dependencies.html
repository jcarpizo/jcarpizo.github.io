Dependencies¶

Danger

Don't commit the dependencies (vendor folder) to the SVN-System!

  • All dependencies MUST be managed by composer.
  • Dependencies MUST NOT be installed manually!
  • DEV/Master-Branches versions MUST NOT be used
  • The Caret (^) operator SHOULD be used for versions

Danger

Dependencies that are not listed here, SHOULD only be used after consultation!
