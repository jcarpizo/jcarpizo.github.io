Project-Setup¶

Download the latest version of the project template:

  • For Web-Applications: https://bitbucket.org/FFUF/ffuf-webapp/downloads
  • For REST-Applications: https://bitbucket.org/FFUF/ffuf-restapp/downloads

Customize¶

Open the composer.json file and change following parts:

APPLICATION_NAME must be replaced with the project name. Naming schema:

  • Valid Characters: [A-Z, a-z, -, 0-9]
  • Blanks becomes dashes (-)
  • No Umlauts (change ä to ae and so on)
  • All letters are lower case
{
    "name": "ffuf/APPLICATION_NAME"
}

Change the author information

{
    "authors": [
        {
            "name": "Firstname Lastname",
            "email": "fistname.lastname@ffuf.de"
        }
    ]
}

Change the namespace for your project. Naming schema:

  • Valid Characters: [A-Z, a-z, 0-9]
  • No Umlauts (change ä to ae and so on)
{
    "autoload": {
        "psr-4": {
            "ffuf\\customer\\project\\": ""
        }
    }
}

Install the Dependencies¶

Run the composer CLI-Command:

composer.phar install --no-dev

Install the Front-End-Dependencies¶

Runt the CLI-Command

bower-installer
