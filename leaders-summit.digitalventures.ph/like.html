Likes¶

Add new like¶

POST /api/likes/(string: post_type)/(int: post_id)/add¶
Parameters:
  • post_type (string) – topic | comment
  • post_id (integer) – the post_id
  • role (string) – participant | user
  • participant_id (integer) – the participant Id

Example response:

HTTP/1.1 200 OK
Vary: Accept
Content-Type: application/json; charset=UTF-8

[
      {
        "response_code": 1000,
        "data": {
          "likes": {
            "id": "368",
            "user_id": "1",
            "post_id": "138",
            "type": "topic",
            "role": "participant"
          }
        },
        "system": {
          "created_at": "2017-01-23 10:28:30",
          "signature": "22a017e845d5ac693009f434cd88190317b59255",
          "message": "Success"
        }
      }
]
